# 图片插入逻辑详解

## 问题背景

用户要求：
1. 图片应该在"如下图"后面的图名上方插入
2. 不要单独的图片列表
3. 不要注意事项

## 模板结构

**正确做法**：模板只包含heading和content，不包含图片列表和注意事项

```
{% for section in sections %}
{{section.heading}}      ← Heading 1
{{section.content}}      ← 正文内容（包含图名和注意事项）
{% for sub in section.subsections %}
{{sub.heading}}          ← Heading 2
{{sub.content}}          ← 子章节内容
{% endfor %}
{% endfor %}
```

**错误做法**：模板中包含图片列表和注意事项循环

```
{% for img in section.images %}    ← 不要这个
{{img.ref}} {{img.description}}
{% endfor %}
{% for note in section.notes %}    ← 不要这个
注意：{{note}}
{% endfor %}
```

## 图片插入逻辑

### 关键点

1. **图名在content字段中**：图名（如"图8 系统总体架构图"）是写在content字段中的，不是独立的段落
2. **一个段落可能包含多个图名**：如"如下图。\n图8 xxx\n\n...如下图。\n图9 xxx"
3. **需要在每个图名前插入图片**：不能只插入一次

### 正确的插入方法

```python
import re
from docx import Document
from docx.shared import Inches

def insert_images(doc_path, image_mapping, output_path):
    """
    在文档中插入图片
    
    image_mapping: {图号: 图片路径}
        例如: {"图8": "diagrams/系统总体架构图.png"}
    """
    doc = Document(doc_path)
    
    # 遍历所有段落，查找包含图名的段落
    paragraphs = list(doc.paragraphs)
    insert_count = 0
    inserted_refs = set()
    
    for i, para in enumerate(paragraphs):
        text = para.text.strip()
        
        # 查找段落中所有图名（如"图8"、"图9"、"图16"）
        fig_refs = re.findall(r'图\d+', text)
        
        for fig_ref in fig_refs:
            if fig_ref in image_mapping and fig_ref not in inserted_refs:
                img_path = image_mapping[fig_ref]
                
                # 在当前段落后插入图片
                # ⚠️ 不能用 doc.add_paragraph()，那会在文档末尾添加
                # ⚠️ 必须用 insert_paragraph_after() 在正确位置插入
                
                new_para = insert_paragraph_after(para)
                new_para.alignment = 1  # CENTER
                run = new_para.add_run()
                run.add_picture(img_path, width=Inches(5.0))
                
                insert_count += 1
                inserted_refs.add(fig_ref)
                print(f'插入 {fig_ref} -> {os.path.basename(img_path)}')
    
    doc.save(output_path)
    print(f'插入图片: {insert_count}张')

def insert_paragraph_after(paragraph):
    """在指定段落后插入新段落"""
    from docx.oxml import OxmlElement
    new_p = OxmlElement('w:p')
    paragraph._element.addnext(new_p)
    from docx.text.paragraph import Paragraph
    new_para = Paragraph(new_p, paragraph._parent)
    return new_para
```

### 错误方法

```python
# ❌ 错误：在文档末尾添加
new_para = doc.add_paragraph()

# ❌ 错误：在图名独立段落前插入（图名不是独立段落）
new_para = para.insert_paragraph_before("")
```

## 完整流程

1. **生成架构图JSON**：使用excalidraw skill
2. **渲染为PNG**：使用render_excalidraw.py（300dpi）
3. **更新content.json**：为images添加path字段
4. **生成文档**：使用generate_manual.py（新模板，无图片列表）
5. **插入图片**：使用上述insert_images函数

## content.json中的images字段

需要为有图片的images添加path字段：

```json
{
  "heading": "（二）总体架构",
  "content": "...如下图。\n图8 系统总体架构图\n\n...",
  "images": [
    {"ref": "图8", "description": "系统总体架构图", "path": "系统总体架构图.png"},
    {"ref": "图9", "description": "前端技术架构", "path": "前端技术架构.png"}
  ]
}
```

## 注意事项

1. **精确匹配图号**：使用 `re.findall(r'图\d+', text)` 提取图号
2. **避免重复插入**：使用 `inserted_refs` 集合记录已插入的图号
3. **图片居中**：设置 `new_para.alignment = 1`（CENTER）
4. **图片宽度**：使用 `Inches(5.0)` 或 `Inches(5.5)`，适合A4纸
