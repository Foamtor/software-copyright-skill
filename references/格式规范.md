   222|    74|for run in h1.runs:
   223|    75|    run.font.name = '黑体'
   224|    76|    run.font.size = Pt(16)
   225|    77|    run.bold = False
   226|    78|    run._element.rPr.rFonts.set(qn('w:eastAsia'), '黑体')
   227|    79|
   228|    80|# 设置二级标题（加粗）

   282|    45|## 写作要点
   283|    46|
   284|    47|1. **描述用户动作**：点击什么按钮、填写什么内容、选择什么选项
   285|    48|2. **描述系统响应**：弹出什么对话框、显示什么页面、提示什么信息
   286|    49|3. **操作顺序**：按用户操作的先后顺序组织
   287|    50|4. **截图配合**：每段操作后跟"如下图。"，然后是"图X xxx示意"
   288|    51|5. **语言简洁**：面向普通用户，不使用技术术语
   289|    52|6. **每段独立**：每个操作步骤是一个独立的段落
