# 示例项目

这是一个示例项目结构，用于演示 `software-copyright-skill` 的使用。

## 使用方法

1. 将此目录替换为你的实际项目
2. 在AI Agent中说："帮我为当前项目生成软著文档"
3. 按照Agent引导完成信息确认和文档生成
